  ИНСТРУКЦИЯ ПО НАСТРОЙКЕ СЕРВЕРА CHILDWATCH
  Простой способ через PuTTY + WinSCP
═══════════════════════════════════════════════════════════════

📌 ШАГ 1: ПЕРЕУСТАНОВКА СЕРВЕРА НА UBUNTU 22.04
──────────────────────────────────────────────────────────────
1. Откройте панель управления Hoster.ru
2. Найдите кнопку "Изменение шаблона VDS" или "Переустановить ОС"
3. Выберите: Ubuntu 22.04
4. Нажмите "ИЗМЕНИТЬ"
5. Подождите 5-10 минут

⚠️ ВАЖНО: После переустановки пароль root может измениться!
   Проверьте email от Hoster.ru или панель управления.


📌 ШАГ 2: СКАЧАТЬ WINSCP (если нет)
──────────────────────────────────────────────────────────────
1. Скачайте WinSCP: https://winscp.net/eng/download.php
2. Установите программу
3. WinSCP - это "проводник" для сервера (как Total Commander)


📌 ШАГ 3: ЗАГРУЗИТЬ СКРИПТ НА СЕРВЕР
──────────────────────────────────────────────────────────────
1. Запустите WinSCP
2. В WinSCP введите:
   ├─ Протокол: SFTP
   ├─ Хост: 31.28.27.96
   ├─ Порт: 22
   ├─ Имя пользователя: root
   └─ Пароль: (ваш новый пароль после переустановки)

3. Нажмите "Войти"

4. В WinSCP вы увидите:
   ├─ СЛЕВА: Ваш компьютер (Windows)
   └─ СПРАВА: Сервер (Linux)

5. СЛЕВА откройте папку:
   C:\Users\dr-ed\ChildWatch\server-setup\

6. СПРАВА перейдите в папку:
   /root/

7. Перетащите файл "setup-childwatch.sh" СЛЕВА → НАПРАВО
   (из Windows на сервер)


📌 ШАГ 4: ЗАПУСТИТЬ СКРИПТ ЧЕРЕЗ PUTTY
──────────────────────────────────────────────────────────────
1. Запустите PuTTY

2. В PuTTY введите:
   ├─ Host Name: 31.28.27.96
   └─ Port: 22

3. Нажмите "Open"

4. Введите:
   login as: root
   password: (ваш пароль)

5. В PuTTY появится командная строка. Введите ПОСЛЕДОВАТЕЛЬНО:

   chmod +x /root/setup-childwatch.sh

   (нажмите Enter, подождите 1 секунду)

   /root/setup-childwatch.sh

   (нажмите Enter)

6. Скрипт запустится! Будет много текста.
   Подождите 5-10 минут, пока не появится:

   ✓ Installation Complete!


📌 ШАГ 5: ЧТО ДЕЛАТЬ ПОСЛЕ УСТАНОВКИ
──────────────────────────────────────────────────────────────
После завершения скрипта на сервере будет:
✅ Node.js 20
✅ PM2 (менеджер процессов)
✅ Nginx (веб-сервер)
✅ Certbot (для SSL)
✅ Firewall настроен
✅ Папка /var/www/childwatch (для кода)

Следующий шаг - загрузить код ChildWatch backend на сервер.


📌 КОМАНДЫ ДЛЯ ПРОВЕРКИ (в PuTTY)
──────────────────────────────────────────────────────────────
node --version          # Проверить Node.js
npm --version           # Проверить NPM
pm2 --version           # Проверить PM2
nginx -v                # Проверить Nginx
ufw status             # Проверить Firewall
